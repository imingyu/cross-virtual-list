# cross-virtual-list

跨平台、跨框架的虚拟列表组件

## 功能特性

| 状态 | 说明                   |
| ---- | ---------------------- |
| 📌   | 已规划，待开发         |
| 💻   | 开发中                 |
| 🔍   | 已开发完毕，待进行测试 |
| ✅   | 已经测试通过           |

### 核心功能

| 状态 | 特性                         |
| ---- | ---------------------------- |
| 🔍   | 仅渲染视口可见的元素         |
| 🔍   | 支持前后缓冲区               |
| 🔍   | 支持横向滚动                 |
| 🔍   | 支持纵向滚动                 |
| 🔍   | 列表项尺寸不变的虚拟列表     |
| 🔍   | 列表项尺寸动态变化的虚拟列表 |
