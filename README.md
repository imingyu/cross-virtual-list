# cross-virtual-list

跨平台、跨框架的虚拟列表组件

## 功能特性

-   [ ] 核心特性
    -   [x] 仅渲染视口可见的元素
    -   [x] 支持前后缓冲区
    -   [?] 支持横向滚动
    -   [x] 支持纵向滚动
    -   [x] 列表项尺寸不变的虚拟列表
    -   [x] 列表项尺寸动态变化的虚拟列表
-   [ ] 跨平台
    -   [ ] Web
    -   [-] 微信小程序
-   [ ] 跨框架
    -   [ ] Web Component
    -   [ ] React
    -   [ ] Vue
    -   [ ] Other
